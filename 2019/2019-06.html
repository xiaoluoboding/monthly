# 📖 2019-06

# 星推

  • pickr (opens new window)

    Flat, Simple, Hackable Color-Picker.

  • sunglass (opens new window)

    给定图像 & 一组颜色，生成按颜色渲染的图像，就像项目名称 sunglass 一样，戴着太阳镜给眼前的色彩加了滤镜。

  • Caprine (opens new window)

    一个优雅的第三方 Facebook Messenger 桌面应用，基于 Electron 构建。

# 工具

  • wkhtmltopdf (opens new window)

    Convert HTML to PDF using Webkit (QtWebKit)

  • Responsive Image Breakpoints Generator (opens new window)

    给定图片，帮你生成响应式的最佳尺寸图片

  • ColorKitty (opens new window)

    给定图片，生成图片中相应色板

  • cfonts (opens new window)

    让你打印在控制台的字体变得生动起来

    同类型工具 chalk (opens new window)

  • alita (opens new window)

    京东多端融合技术部出品的一套把React Native代码转换成微信小程序代码的转换引擎工具

  • 在线PS工具 (opens new window)

    免安装的在线PS工具，可以满足一些PS场景，甚至更繁琐的操作

# 教程

  • 现代 JavaScript 教程中文版 (opens new window)

  • VS Code can do that?! (opens new window)

    用短视频的方式介绍 VS Code 一些你也许不知道的小技巧，目前已收录35个。

  • Animating SVG with CSS (opens new window)

    多个简单小例子讲解如何利用CSS让SVG动起来

  • HTML can do that? (opens new window)

    看看用 HTML 能实现哪些实用的功能，目前收录了10个小例子。

# 阅读

  • State of CSS 2019 (opens new window)

    回顾一下 2019 年度哪些 CSS 特性用的最多，哪些工具正在被采用。

  • VueConf 2019 上海演讲全部资料 (opens new window)（含视频）

    尤大的演讲 **State of Vue，**讲述 Vue 现状和3.0发展以及新的rfcs，Vue3.0 会变得更快更好，迫不期待上手 Vue 3.0。

# 应用

  • Motrix (opens new window)

    一款免费开源的全能的下载工具，支持下载 HTTP、FTP、BT、磁力链、百度网盘等资源

  • Patatap (opens new window)

    上班摸鱼，很有趣的小游戏，能玩一整天的洗脑旋律，用键盘发现动感节拍

    同类型应用 mikutap 初音版传送门 (opens new window)

  • magickeyboard.io (opens new window)

    上班摸鱼，hacker 风格的解压网站，输入的字符会弹起，模拟机械键盘音效。

  • Microsoft To-Do (opens new window)

    一直在用奇妙清单这款应用，我用它记录了很多整理过的清单、Todo列表。不过奇妙清单也归加入了微软，最近微软推出了 Mircosoft To-Do 可以导入奇妙清单的数据，这是一款未来要代替奇妙清单的 ToDo 应用。
