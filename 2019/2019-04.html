# 📖 2019-04

# 动态

  • Helvetica 发布了 Helvetica Now 字体 (opens new window)
  • JQuery 发布了 3.4.0 版本 (opens new window)
  • Electron 发布 5.0.0 稳定版 (opens new window)

# 星推

  • Squoosh! (opens new window)

    Squoosh is an image compression web app that allows you to dive into the advanced options provided by various image compressors.

  • ffsend (opens new window)

    Easily and securely share files from the command line. A fully featured Firefox Send client.

  • Cube.js (opens new window)

    一个用于构建分析 Web 应用程序的开源框架

  • docker-node (opens new window)

    Node.js 官方 Docker 镜像

# 工具

  • JavaScript Coercions Grid (opens new window)

    JavaScript 强制类型转换表格

  • SVG在线压缩合并工具 (opens new window)

    张鑫旭大神打造的小工具，底层技术使用 svgo。

  • Capture API (opens new window)

    Capture 提供了免费的页面生成PDF与截屏API

  • dbdiagram (opens new window)

    数据库关系图构建工具，对程序员比较友好，用写代码的方式即可构建数据库关系图。

  • BundlePhobia (opens new window)

    一个分析npm包大小以及性能的可视化工具，帮助你在引入模块前做一些决策。

  • beautiful.ai (opens new window)

    AI 加持的创建演示文稿应用

  • Python Data Science Handbook (opens new window)

  • Everything you need to know about Loading Animations (opens new window)

    没有人喜欢等待，用 Loading 动画为你的应用提升用户交互体验吧

  • JavaScript Grammar (opens new window)

    免费获得 Greg Sidelnikov 的电子书，只需关注作者的twitter以及留下你的邮箱即可

  • The Book of Shaders (opens new window)

    一本关于 Fragment Shaders (片段着色器) 的入门指南

  • 30 days of react (opens new window)

    JS 模块化简史

  • 5 steps for including motion design in your system (opens new window)

  • WebAssembly 现状与实战 (opens new window)

# 创意

  • Hanzi Writer (opens new window)

    外国友人做的汉字比划顺序书写动画，你以为你了解比划顺序嘛，来看看。

  • Emojipedia (opens new window)

    Emoji 百科

  • CaptionBot.ai (opens new window)

    CaptionBot.ai 使用机器学习技术来识别和描述你的图片。当你上传一张图片，它会被发送给Microsoft进行分析，然后返回一句话的描述。

# 应用

  • Notion (opens new window)

    重新定义了笔记软件，可作为团队写作工具、私人知识库、文档中心、任务管理等等工具使用，我主要用来同步笔记，本期月刊是用它编辑的。

  • Moon FM (opens new window)

    比较小众的FM桌面版，同时也ios版，界面美观优雅。
