# 📖 2022-05

聚焦前端，记录过去一个月发现的值得推荐的前端技术栈、文章、教程以及应用，每月28日更新，本期第四十一期。2022 年度第五期。

# 星推

Github Star 推荐

  • 🔧 markdoc

    Strip 上周开源的文档工具，一周吸了 3200+ star，提供了 Markdown 文本转换 AST，AST 转换 content 等 API，支持渲染为 HTML、React

# 工具

  • 🔧 JSON Hero (opens new window)

    JSON Hero 是最近 star 数量猛增的一款可视化工具，甚至是 IDE 工具，支持多种数据类型的可视化，比如：

    • 图像
    • 日期
    • 颜色
    • 链接
    • 视频

    提供了类似于 KBar 的搜索框，可快速定位字段，同时提供了类似 macOS Finder 类似的多面板探索界面

  • 🔧 cover-view (opens new window)

    用于创建媒体宣传封面的小工具

  • 🔧 URL shortener (opens new window)

    可自部署的短链接服务

  • 🔧 spacedrive (opens new window)

    一款开源的面向未来的跨平台文件管理软件

  • 🔧 samizdatco/skia-canvas (opens new window)

    Node.js 下的 Canvas 渲染解决方案

  • 🔧 taskforcesh/bullmq (opens new window)

    Node.js 下的一款基于 Redis 的快速而强大的队列系统，有助于解决许多现代的微服务架构。

  • 🔧 google/wireit (opens new window)

    更有效的使用 NPM 命令

  • 🔧 Snipli (opens new window)

    将本地文件转换为 Gist 的小工具

  • 🔧 contentlayerdev/contentlayer (opens new window)

    将 MD（X）、Notion 等内容转换为开发者友好的 JSON 格式数据，适用于构建 CMS 内容，目前支持框架有：

    • Next.js
    • Vite
    • Remix
    • Astro

  • 🔧 Pulumi (opens new window)

    Pulumi 是一个现代 infrastructure as code (opens new window) 平台，允许您使用熟悉的编程语言和工具来构建、部署和管理云基础架构。

  • 🔧 OneDev (opens new window)

    可自部署的 Git 服务，带有 CI/CD、看板功能

  • 🔧 alexmacarthur/striff (opens new window)

    字符串对比类库，对比结果通过 JSON 结构描述

  • 🔧 sindresorhus/clipboardy (opens new window)

    跨系统的剪切板类库

  • 🔧 maptalks/maptalks.js (opens new window)

    轻量可插拔的 2d/3d 地图类库

  • 🔧 gitbrent/PptxGenJS (opens new window)

    用于创建 PowerPoint 的 JavaScript API

  • 🔧 zernonia/tweetic (opens new window)

    将 Twitter 转换渲染为分享卡片的小工具

  • 🔧 Code to Image (opens new window)

    一个将代码转换为美观分享图的小工具

  • 🔧 Design Tools Database (opens new window)

    设计工具数据库，业内最好的设计工具供您选择

  • 🔧 React Static (opens new window)

    ⚛️ 🚀 基于 React 的 JAMStack 静态站点生成器

  • 🔧 sachinchoolur/lightGallery (opens new window)

    轻量的图片灯箱（lightbox）插件，一般适用于博客、CMS查看图片组

  • 🔧 baptisteArno/typebot.io (opens new window)

    一款可自部署的开源会话表单构建器

  • 🔧 requarks/wiki (opens new window)

    基于 Node 构建好用的 Wiki 应用

  • 🔧 LOGO设计神器 (opens new window)

    LOGO神器是一款智能LOGO在线设计生成器。只需输入品牌名称就能免费在线生成公司logo设计，商标设计，以及配套企业VI助您打造个性品牌。

    笔者试用了一下，生成的 LOGO 不俗气，完全可以二次加工来使用。

  • 🔧 adobe/leonardo (opens new window)

    Adobe 提供的设计师友好的色彩设计系统，特点是根据对比度生成颜色系统，提供色彩检查工具，让色彩设计在可控的安全范围，轻松创建你的调色盘。

    该工具还可以用于创建用于可视化的色彩主题，比如根据提供的关键色搭配用于渲染地图、柱状图、散点图所需的颜色主题。
