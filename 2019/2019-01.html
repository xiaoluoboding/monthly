# 📖 2019-01

# 星推

Github Star 推荐

  • 2018年 JavaScript 明星项目 (opens new window)

    前10名里，有1个项目为中国区个人项目 Vue Element Admin (opens new window)

  • hacker-news-pwas (opens new window)

    收集了 hacker-news 的若干种 PWA 实现方式，因架构不同而表现出不同的性能

  • clean-code-javascript (opens new window)

    代码整洁之道的 JavaScript 实践，看到自己的代码有些莫名契合

# 工具

  • State of JavaScript 2018 (opens new window)

    2019年初我们回顾下2018年 JavaScript 业界动态

  • percollate (opens new window)

    命令行工具，使用 puppeteer 将网页生成PDF

  • screenshoteer (opens new window)

    命令行工具，使用 puppeteer 将网页生成截图

  • Programmingfonts (opens new window)

    一个收集了常用于编程的免费等宽字体50+，可以在线预览字体效果。我经常使用的是 firacode，这个字体能更好的辨认运算符

  • Data to Viz (opens new window)

    一个不错的图表类型介绍网站

  • 性能优化资源 (opens new window)

    性能优化工具，包括分析、优化、监控、书籍等好用的工具集合

  • Material Design Color Tool (opens new window)

    谷歌 Material Design 官方调色板工具

  • Material Design Palette Generator (opens new window)

    谷歌 Material Design 调色板生成器，由 Kuali 团队提供

  • eleventy (opens new window)

    一个简洁的静态网站生成工具

# 教程

  • [译文]网页动画的十二原则 (opens new window)

    迪士尼经过基础工作练习的长时间累积，发表了动画的十二个原则，作者将这十二个原则付诸实践，我们可以学习如何将原则运用到网页开发中

  • 看见统计 (opens new window)

    看得见的统计概率入门，致力于用数据可视化让统计概念更容易理解。该教程已经整理成书（英文版），相信未来会有中文版

  • How to make VR with the web, a new video series (opens new window)

    作者通过7个简短的视频教你如何用 three.js 和 WebAR 开发一个AR应用

  • What happens when you type 'google.com' into a browser and press Enter? (opens new window)

    面试中老生常谈的经典面试题，当在浏览器输入 google.com 敲回车时候到底会发生什么，考察面试者的方方面面

  • Browser Rendering Optimizations for Frontend Development (opens new window)

    浏览器渲染性能优化

  • CSS Specificity (opens new window)

    最近 dev.to 社区比较火的程序媛小姐姐 Emma 教你什么是 CSS Specificity

  • Front-End Performance Checklist 2019 (opens new window)

    涵盖了目前你所需要知道的关于打造极速体验的所有内容，阅读中文版 (opens new window)

  • The TypeScript Tax (opens new window)
